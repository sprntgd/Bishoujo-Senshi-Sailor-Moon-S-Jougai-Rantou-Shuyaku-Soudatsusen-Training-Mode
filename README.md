Training mode and color edit mods for Bishoujo Senshi Sailor Moon S: Jougai Rantou!? Shuyaku Soudatsusen released for the Super Famicom on December 16, 1994 and Bishoujo Senshi Sailor Moon Super S: Zenin Sanka!! Shuyaku Soudatsusen released for the Super Famicom on March 29, 1996.

The [Lua script](SailorMoonS.lua?raw=1) adds training features and color edit mode to the game when emulated in BizHawk and Snes9X.  
For more details on the Lua script and the emulators required see [readme-lua.txt](readme-lua.txt?raw=1)

The patcher modifies an original game ROM to add custom colors and some other useful features.  
For more details on the patcher see [readme-patcher.txt](readme-patcher.txt?raw=1)  
(Note: The patcher does not currently support Zenin Sanka)
